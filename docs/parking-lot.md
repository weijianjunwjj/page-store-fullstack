# Parking Lot

> 所有「顺便想做」「先记下来以后再说」的想法都进这里,**绝对不当天做**。
> 进入这个文件 = 暂时不做。**写下来的目的就是让自己安心放下。**

---

## 使用规则

1. 想到立刻写,**不超过 30 秒**(写多了就是在做这件事)
2. 每条只用一行:`- [YYYY-MM-DD] 想做的事情一句话`
3. **不许**写「为什么想做」「怎么做」「优先级」——多写一个字都是延期触发器
4. 每周日晚上 5 分钟看任务卡时,**顺便**扫一眼这个文件
5. **W3 结束 / W6 结束 / W8 结束** 三个节点,主动 review parking lot,决定哪些进 v2
